# 문제 제작 지식 베이스

## 1. 인적성 문제 패턴

### GSAT 언어추리 패턴
**인과관계 문제**
- "가장 직접적인 영향은?"
- "1차적 결과는?"
- "즉각적 변화는?"

**선택지 구성**
- ① 1차 효과 (정답)
- ② 2차 효과 (매력적 오답)
- ③ 3차 효과
- ④ 무관한 내용

### SKCT 상황판단 패턴
**Trade-off 문제**
- "우선적으로 고려할 사항은?"
- "가장 중요한 이유는?"
- "최적의 선택은?"

**선택지 구성**
- ① 핵심 논리 (정답)
- ② 부차적 이유
- ③ 반대 입장
- ④ 감정적 판단

### 자료해석 공통 패턴
**빈칸 추론**
- 맥락상 유일한 답
- 앞뒤 문장과 연결
- 경제지표 정확성

**데이터 해석**
- 수치의 의미 파악
- 변화 방향 이해
- 인과관계 추론

## 2. 경제 주제별 문제 템플릿

### 통화정책 관련
**블랙스완 체인**
금리 변동 → 시중금리 → 대출/예금 → 소비/투자 → 경기

**딜레마 구조**
물가안정 vs 경제성장
긴축 vs 완화

**핵심 용어**
기준금리, 양적완화, 테이퍼링, 역레포

### 부동산 관련
**블랙스완 체인**
규제 변화 → 대출 한도 → 매수 심리 → 거래량 → 가격

**딜레마 구조**
주거안정 vs 시장기능
규제 vs 완화

**핵심 용어**
DTI, DSR, LTV, 전세가율

### 주식시장 관련
**블랙스완 체인**
기업 실적 → 주가 → 시총 → 인덱스 → 자금 흐름

**딜레마 구조**
성장주 vs 가치주
단기 vs 장기

**핵심 용어**
PER, PBR, ROE, EPS, 배당수익률

## 3. 선택지 함정 패턴

### 블랙스완 함정
1. **시간차 함정**: 나중에 일어나는 일을 먼저로
2. **인과 역전**: 원인과 결과 바꾸기
3. **과잉 일반화**: 특수한 경우를 일반화
4. **생략 오류**: 중간 단계 건너뛰기

### 딜레마 함정
1. **입장 혼동**: A의 논리를 B에게 적용
2. **감정 오류**: 논리 대신 감정적 판단
3. **극단 선택**: 중도적 입장 무시
4. **허수아비**: 없는 논리 만들기

### 시그널 함정
1. **유사 용어**: 비슷하지만 다른 개념
2. **단위 오류**: %와 %p 혼동
3. **시점 오류**: 전년비, 전월비 혼동
4. **범위 오류**: 부분과 전체 혼동

## 4. 해설 작성 공식

### 3문장 구조
1문장: 정답 근거 (기사 팩트)
2문장: 인적성 연계 (시험 스킬)
3문장: 실무 활용 (실전 팁)

### 문장별 시작 어구
1문장: "기사에 따르면~", "~이 핵심"
2문장: "GSAT/SKCT에서는~", "인적성 관점에서~"  
3문장: "실무적으로~", "실제로는~"

## 5. 난이도 조절 기법

### 상(★★★)
- 3단계 이상 추론 필요
- 선택지 간 미묘한 차이
- 복합적 사고 요구
- 시간 압박 고려

### 중(★★☆)
- 2단계 추론
- 명확한 구분점
- 표준적 사고
- 적정 시간

### 하(★☆☆)
- 1단계 직접 연결
- 명백한 정답
- 기초 개념
- 빠른 해결

## 6. 최종 검증 체크리스트

### 문제 독립성
✓ 기사 없이 읽어도 이해되는가?
✓ 모든 고유명사가 명시되었는가?
✓ 지시대명사가 없는가?

### 정답 명확성
✓ 정답이 논란 없이 명확한가?
✓ 기사에서 근거를 찾을 수 있는가?
✓ 인적성 로직과 일치하는가?

### 교육 효과
✓ 실제 시험에 도움되는가?
✓ 경제 상식이 늘어나는가?
✓ 사고력이 향상되는가?


## 7. 암묵적 효과 전달 방법

### 직접 언급 금지, 간접 효과 명시
**절대 표현 금지:**
- "GSAT/SKCT/HMAT 대비"
- "인적성 직결/필수"
- "취업/합격 보장"
- "출제 예상"

**대신 사용:**
- "논리적 사고 훈련"
- "체계적 분석 능력"
- "실무 판단력 향상"

### 게임별 핵심 역량 (내부 참고용)
| 게임 | 표면적 목적 | 실제 훈련 | 출력 표기 |
|------|------------|----------|----------|
| 블랙스완 | 경제 연쇄반응 이해 | 인과추론 (GSAT형) | 🌊 블랙스완 - 연쇄반응 분석 |
| 딜레마 | 균형적 판단 | 상황판단 (SKCT형) | ⚖️ 죄수의 딜레마 - 균형적 판단 |
| 시그널 | 데이터 해석 | 자료해석 (HMAT형) | 🔍 시그널 디코딩 - 데이터 해석 |

## 8. 정답 분포 관리

### 정답 분포 관리

#### 필수 규칙
- ①번 사용: 4문제당 1회 이하
- 당일 3문제: 서로 다른 번호
- 우선순위: ②③④번 적극 활용

#### 일별 정답 배치 예시
월: 블랙스완② 딜레마④ 시그널③
화: 블랙스완③ 딜레마② 시그널④
수: 블랙스완④ 딜레마③ 시그널②

### 해설 작성 공식

#### 최종 출력 (서술식)
- 3문장으로 작성 (라벨 없이)
- 첫 문장: [정답 설명] 내용을 서술식으로
- 둘째 문장: [경제 원리] 내용을 자연스럽게
- 셋째 문장: [시사점] 내용을 연결하여

#### 내부 체크용 (출력하지 않음)
[카테고리] 25자 이내 - 내부 메모용

## 9. 문법/맞춤법 체크포인트

### 자주 틀리는 조사
- 피벗을 (O) / 피벗를 (X)
- GDP가 (O) / GDP이 (X)
- HBM이 (O) / HBM가 (X)

### 경제 용어 띄어쓰기
- 기준금리 (O) / 기준 금리 (X)
- 소비자물가지수 (O) / 소비자 물가지수 (X)
- 고대역폭 메모리 (O) / 고대역폭메모리 (X)

## 10. 파일 우선순위 및 충돌 방지

### 출력 시 우선순위
1. master_output_format.xml (형식)
2. sample_articles_news.xml (예시)
3. 본 날리지 파일 (로직)

### 충돌 시 해결
- 형식 관련: master_output_format.xml 우선
- 품질 관련: sample_articles_news.xml 참조
- 로직 관련: 본 파일 참조
