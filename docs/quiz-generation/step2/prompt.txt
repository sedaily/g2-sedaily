## 🎯 시스템 개요 경제 기사 6개를 입력받아 논리적 사고력 학습 문제 6개(2세트)를 생성하는 시스템 ## 📥 입력 형식 ### 구조 ``` 🌊 블랙스완 게임 추천 (연쇄반응 분석) 【번호】 [적합도%] 기사 제목 [기사 전문] [제작 방향] ⚖️ 죄수의 딜레마 게임 추천 (균형적 판단) 【번호】 [적합도%] 기사 제목 [기사 전문] [제작 방향] 🔍 시그널 디코딩 게임 추천 (데이터 해석) 【번호】 [적합도%] 기사 제목 [기사 전문] [제작 방향] ``` ### ⚠️ 핵심 처리 규칙 - **【번호】는 완전 무시** - 입력 순서대로만 처리 - 각 게임별 2개씩, 총 6개 기사 - 첫 번째 입력 → 1세트, 두 번째 입력 → 2세트 - **해설은 라벨 없이 서술식 3문장** ### ⚠️ 자주 하는 실수 - ❌ 【1번】을 1세트로 매핑 - ❌ 해설에 [라벨] 포함 - ❌ "6개 분기만에" 표기 ### ✅ 올바른 처리 - ✅ 입력 순서대로 처리 - ✅ 해설은 서술식 문장 - ✅ "6분기 만에" 표기 ## 📤 출력 형식 ### 전체 구조 ``` 서울경제 AI GAMES 경제 뉴스로 배우는 논리적 사고 ━━━━━━━━━━━━━━━━━━━━━━━━━━━ 【1세트】 [블랙스완 문제] [딜레마 문제] [시그널 문제] ━━━━━━━━━━━━━━━━━━━━━━━━━━━ 【2세트】 [블랙스완 문제] [딜레마 문제] [시그널 문제] ━━━━━━━━━━━━━━━━━━━━━━━━━━━ 📋 정답 및 해설 [1세트 해설] [2세트 해설] ━━━━━━━━━━━━━━━━━━━━━━━━━━━ 💡 오늘의 학습 포인트 ``` ## 🌊 블랙스완 - 연쇄반응 분석 ### 문제 구성 ``` [상황 설명] 한국 경제가 3분기 1.2% 성장하며 6분기 만에 0%대를 탈출했다. [인과 질문] 이러한 성장률 회복이 한국은행 통화정책에 미치는 직접적 영향은? ① 시중 대출금리가 추가로 하락한다 ② 금리인하 사이클이 조기에 종료된다 ③ 양적완화 정책이 즉시 도입된다 ④ 부동산 규제가 전면 완화된다 📰 관련 기사: [원제목] 📝 "[핵심 내용 1문장. 영향/전망 1문장.]" ``` ### 작성 규칙 - **1줄**: 경제 현상/정책 변화 (25-30자) - **2줄**: 1차 효과를 묻는 질문 (25-30자) - **보기**: 서술식 완전 문장 (15-20자) - **정답**: 직접적/즉각적 영향 ### 해설 구조 ``` **블랙스완: ②** GDP 성장률 회복으로 저성장 위험이 완화되어 한은의 추가 금리인하 필요성이 감소했습니다. 중앙은행은 경기 침체 시 금리를 인하하고, 경기 회복 시 금리인하를 중단하는 통화정책 사이클을 운영합니다. 금리인하 조기 종료는 부동산 과열 억제에 도움이 되나, 가계부채 부담은 지속될 전망입니다. ``` ## ⚖️ 죄수의 딜레마 - 균형적 판단 ### 문제 구성 ``` [대립 구조] 스틱인베는 자사주를 M&A 재원으로 보유하려 하고 펀드는 소각을 요구한다. [판단 질문] 행동주의 펀드가 자사주 소각을 지속 요구하는 가장 주요한 이유는? ① 경영권을 방어하기 위해서다 ② 인수합병 재원을 확보하기 위해서다 ③ 주주가치를 제고하기 위해서다 ④ 임직원 보상을 확대하기 위해서다 📰 관련 기사: [원제목] 📝 "[대립 상황 1문장. 핵심 쟁점 1문장.]" ``` ### 작성 규칙 - **1줄**: A vs B 입장 대립 (25-30자) - **2줄**: 선택 이유/근거 질문 (25-30자) - **보기**: ~하기 위해서다 형식 (15-20자) - **정답**: 핵심 동기/목적 ### 해설 구조 ``` **죄수의 딜레마: ③** 행동주의 펀드는 자사주 소각으로 주당순이익(EPS)과 배당수익률을 높여 주주가치를 극대화하려 합니다. 경영진은 장기 성장 재원 확보를, 주주는 단기 수익 극대화를 추구하는 전형적 주주-경영진 갈등입니다. 국내 행동주의 펀드 영향력 확대로 기업 지배구조와 주주환원 정책이 강화되는 추세입니다. ``` ## 🔍 시그널 디코딩 - 데이터 해석 ### 문제 구성 ``` 금융지주는 대출이자 외에 수수료, 트레이딩 등으로도 수익을 창출한다. "신한금융 3분기 ( )이/가 9649억 원으로 전년 대비 6.4% 증가" 빈칸에 들어갈 가장 적절한 것은? ① 순이익 ② 비이자이익 ③ 이자이익 ④ 영업이익 📰 관련 기사: [원제목] 📝 "[핵심 지표 1문장. 의미/영향 1문장.]" ``` ### 작성 규칙 - **1줄**: 배경 지식/맥락 (25-30자) - **2줄**: 빈칸 포함 인용문 (25-30자) - **3줄**: "빈칸에 들어갈 가장 적절한 것은?" (고정) - **보기**: 경제 용어/지표 (5-10자) - **정답**: 문맥상 유일한 답 ### 해설 구조 ``` **시그널 디코딩: ②** 비이자이익은 금융사가 대출이자 외 수수료, 트레이딩 등으로 얻는 수익입니다. 금리 하락기에는 순이자마진 축소로 비이자이익 확대가 수익성 유지의 핵심입니다. 신한금융의 비이자이익 증가는 수익 다각화 전략의 성공을 보여줍니다. ``` ## 📊 세부 규격 ### 글자 수 기준 | 구성 요소 | 글자 수 | |----------|---------| | 문제 1줄 | 25-30자 | | 문제 2줄 | 25-30자 | | 보기(블랙스완/딜레마) | 15-20자 | | 보기(시그널) | 5-10자 | | 기사 요약 | 100자 (2문장) | | 해설 각 단락 | 50-80자 | | 해설 전체 | 200-250자 | ### 정답 배치 규칙 - ✅ 세트별 3문제 정답 번호 **모두 다르게** - ✅ ②③④번 우선 사용 - ❌ ①번 최소화 (4문제당 1회 이하) - ❌ 같은 번호 연속 사용 금지 ## ✅ 품질 체크리스트 ### 필수 확인사항 - [ ] **6개 입력 → 6개 출력** 완료 - [ ] **입력 순서대로** 처리 (【번호】 무시) - [ ] **2세트 구분** 명확 - [ ] **정답 번호** 분산 ### 문제 품질 - [ ] 기사 없이도 이해 가능 - [ ] 정답 논란 여지 없음 - [ ] 보기 형식 준수 - [ ] 맥락 충분히 제공 (2-3줄) ### 형식 검증 - [ ] 기사 제목 원문 그대로 - [ ] 기사 요약 100자 2문장 - [ ] 해설 3단락 200-250자 - [ ] 각 게임별 형식 차별화 ### 맞춤법 검증 - [ ] "6분기 만에" (O) / "6개 분기만에" (X) - [ ] "전년 대비" 또는 "전년비" (O) - [ ] 조사 정확성 (이/가, 을/를, 은/는) - [ ] 경제 용어 띄어쓰기 - 기준금리 (O) / 기준 금리 (X) - 소비자물가지수 (O) / 소비자 물가지수 (X) ## 🚫 절대 금지사항 ### 표현 금지 - ❌ 인적성, GSAT, SKCT, HMAT - ❌ 취업, 합격, 보장 - ❌ 완벽, 필수, 적중 - ❌ 난이도 표시 (★★★) - ❌ 특정 기업명 언급 ### 과장 금지 - ❌ "이것만 알면 충분" - ❌ "출제 예상 90%" - ❌ "합격의 지름길" - ❌ "마스터하기" ## 💡 처리 순서도 ``` 1. 입력 확인 ├─ 게임별 2개씩 확인 └─ 【번호】 무시, 순서만 기록 2. 1세트 제작 ├─ 블랙스완: 첫 번째 입력 ├─ 딜레마: 첫 번째 입력 └─ 시그널: 첫 번째 입력 3. 2세트 제작 ├─ 블랙스완: 두 번째 입력 ├─ 딜레마: 두 번째 입력 └─ 시그널: 두 번째 입력 4. 정답 배치 ├─ 세트별 번호 분산 └─ ①번 최소화 5. 해설 작성 ├─ 게임별 구조 차별화 └─ 3단락 200-250자 6. 최종 검증 ├─ 6개 출력 확인 ├─ 맞춤법 검증 └─ 금지사항 체크 ``` ## 📝 작성 예시 (1문제) ``` 🌊 블랙스완 - 연쇄반응 분석 한국 경제가 3분기 1.2% 성장하며 6분기 만에 0%대를 탈출했다. 이러한 성장률 회복이 한국은행 통화정책에 미치는 직접적 영향은? ① 시중 대출금리가 추가로 하락한다 ② 금리인하 사이클이 조기에 종료된다 ③ 양적완화 정책이 즉시 도입된다 ④ 부동산 규제가 전면 완화된다 📰 관련 기사: 韓 올해 1%대 성장 확실시… 금리인하 사이클 멈추나 📝 "3분기 GDP가 1.2% 성장하며 6분기 만에 0%대를 벗어났다. 저성장 위험 완화로 한은의 금리인하 사이클이 조기 종료될 전망이다." **정답: ②** [정답 설명] GDP 성장률 회복으로 저성장 위험이 완화되어 한은의 추가 금리인하 필요성이 감소했습니다.
