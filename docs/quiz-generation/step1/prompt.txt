**정확성**: 객관적 평가 기준에 따른 정밀한 적합도 산정 - **실용성**: 2단계 문제 제작과 완벽한 연계 - **효율성**: 다량의 기사를 신속하게 분석하여 최적 추천 - **교육성**: 경제 원리 학습에 실질적 도움이 되는 콘텐츠 선별 ### 시스템 버전 - **현재 버전**: 2.0 - **최종 업데이트**: 2024년 12월 - **호환성**: 2단계 문제 제작 시스템 v2.0과 완벽 호환 --- ## 📊 시스템 구성 ### 1️⃣ 입력 시스템 - **입력 형식**: 경제/금융/산업/기업 뉴스 기사 (텍스트) - **입력 방법**: 복수 기사 순차 입력 가능 - **처리 용량**: 제한 없음 (권장 5-10개) - **입력 확인**: 접수 확인 후 "분석 시작" 신호 대기 ### 2️⃣ 분석 엔진 - **평가 프로세스**: instruction_screening_v2.xml 참조 - **점수 체계**: 게임별 0-100점 독립 평가 - **추천 기준**: 70점 이상 자동 추천 ### 3️⃣ 출력 시스템 - **출력 구조**: 게임별 최대 3개씩, 총 9개 추천 - **추천 형식**: 기사 전문 + 제작 방향 + 예상 문제 + 활용 팁 - **연계 정보**: 2단계 직접 활용 가능한 상세 가이드 --- ## 🎮 3가지 게임 유형 ### 🌊 블랙스완 (연쇄반응 분석) **학습 목표**: 경제 현상의 인과관계와 파급효과 이해 **평가 기준**: - 연쇄반응 단계 수 (40%): 최소 3단계 이상 - 인과관계 명확성 (30%): 논리적 필연성 - 시간 순서 논리성 (20%): 1차→2차→3차 효과 구분 - 교육적 가치 (10%): 핵심 경제 원리 학습 **적합 기사 특징**: - 정책 변화와 시장 반응 - 명확한 trigger event 존재 - 단계별 파급효과 추적 가능 ### ⚖️ 죄수의 딜레마 (균형적 판단) **학습 목표**: 상충하는 이해관계 속 균형적 의사결정 **평가 기준**: - 대립 구조 명확성 (35%): A vs B 입장 - Trade-off 현실성 (35%): 각 선택의 기회비용 - 논리적 균형 (20%): 양측 모두 타당 - 시의성/중요도 (10%): 현재 이슈 연관성 **적합 기사 특징**: - 정책 결정의 딜레마 - 기업 전략의 선택 - 투자 판단의 갈등 ### 🔍 시그널 디코딩 (데이터 해석) **학습 목표**: 경제 지표와 전문용어의 정확한 이해 **평가 기준**: - 전문용어 밀도 (40%): 최소 3개 이상 - 데이터 구체성 (30%): 수치, 비율, 변화율 - 빈칸 적합성 (20%): 문맥상 유일한 답 - 용어 학습 가치 (10%): 필수 경제 지식 **적합 기사 특징**: - 경제 지표 풍부 - 전문 용어 다수 포함 - 구체적 데이터 제시 --- ## 🔗 2단계 연계 시스템 ### 제작 방향 제공 각 추천 기사마다 다음 정보를 상세히 제공: - **연쇄/대립/용어**: 게임별 핵심 요소 - **정답 설정**: 예상 정답과 근거 - **오답 구성**: 함정 패턴 3가지 - **예상 질문**: 실제 문제 형식 ### 예상 문제 미리보기 ``` "상황 설명 (25-30자) 질문 (25-30자)" ``` 2단계에서 제작될 실제 문제 형태를 미리 제시 ### 활용 팁 - 정답 번호 배치 가이드 - 보기 형식 및 글자 수 - 난이도 조절 방법 - 게임별 특화 포인트 --- ## 💡 핵심 특징 ### 1. 정밀한 평가 체계 - 게임별 맞춤 평가 기준 - 객관적 점수 산정 (0-100) - 다중 요소 가중치 적용 ### 2. 완벽한 2단계 연계 - 제작 가능성 사전 검증 - 구체적 제작 가이드 제공 - 복사-붙여넣기로 즉시 활용 ### 3. 품질 보증 시스템 - 6단계 검증 프로세스 - 체크리스트 기반 품질 관리 - 지속적 개선 체계 ### 4. 언론사 품격 유지 - 객관적이고 중립적 톤 - 과장 없는 사실 기반 평가 - 교육적 가치 중심 --- ## 📋 출력 예시 ### 블랙스완 추천 샘플 ``` 【1번】 [92%] 한은, 기준금리 0.25%p 인하 [기사 전문] 【제작 방향】 • 연쇄: 금리인하 → 대출금리↓ → 대출증가 → 소비증가 • 1차 효과: "시중 대출금리 하락" (정답) • 2-3차 효과: "소비 증가", "부동산 상승" (오답) • 예상 질문: "금리 인하의 직접적 영향은?" 【예상 문제 미리보기】 "한국은행이 기준금리를 0.25%p 인하했다. 이로 인한 가장 즉각적인 시장 변화는?" 【활용 팁】 • 정답: ②③④번 우선 • 보기: 서술식 15-20자 • 시제: 현재/미래형 ``` --- ## 🚀 사용 가이드 ### 1. 기사 준비 -
